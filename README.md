# Portfolio Website

A modern and responsive portfolio website built with PHP, JavaScript, CSS, and PHPMailer. This project showcases my skills, projects, and professional journey while providing visitors with a direct way to get in touch through a contact form.

## Features

* Responsive and mobile-friendly design
* Project showcase section
* Contact form with email notifications
* Automatic email replies using PHPMailer
* Admin dashboard for managing portfolio content
* SEO-ready with robots.txt and sitemap.xml
* Clean and organized project structure

## Tech Stack

* PHP
* JavaScript
* HTML5
* CSS3
* Composer
* PHPMailer

```text
Portfolio/

```

## Installation

1. Clone the repository

```bash
git clone https://github.com/Nestor-dev001/Portfolio.git
```

2. Navigate into the project directory

```bash
cd Portfolio
```

3. Install dependencies

```bash
composer install
```

4. Configure your database and email credentials.

5. Start your local server and access the project through your browser.
